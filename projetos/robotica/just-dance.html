Sobre o projeto

O projeto foi desenvolvido por alunos da turma de CT2 de Sexta as 09:30, onde o objetivo foi recriar o jogo Just Dance usando o scratch. Para isso, os alunos conectaram o makey makey as peças de papelão cobertas com alumínio condutor e construíram a programação sozinhos.

Quem desenvolveu?

Os alunos responsáveis pelo projeto foram: fábio wesley e William, do curso de Ctrl Teens 2. Os meninos colocaram em prática o conteúdo aprendido nas aulas anteriores para elaborarem toda a parte computacional e estrutural

Tecnologias utilizadas

Neste projeto utilizamos o Makey Makey, uma placa de programação para robótica, que controla as respostas das setas.
