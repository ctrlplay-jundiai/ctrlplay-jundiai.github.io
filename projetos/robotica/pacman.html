Pac-Man - Micro:bit

Sobre o projeto

Nesse projeto nossos alunos se inspiraram no clássico jogo e desenvolveram no make code a programação lógica do PacMan, onde temos o fantasma, a comida e o PacMan, em forma de LEDs no micro:bit.

Quem desenvolveu?

Turmas Kids - Diversas

Tecnologias utilizadas

Micro:bit, bateria.
