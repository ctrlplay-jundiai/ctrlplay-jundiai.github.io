Sobre o projeto

O projeto é baseado em um brinquedo de memória criado a muito tempo atrás. Onde o jogador deve lembrar a sequência de botões iluminados e depois apertar a mesma, conforme o jogador acerta a sequência aumenta. Assim o jogo só termina quando o jogador erra. Para a criação desse projeto foi utilizado leads e botões com cores correspondentes, buzzer para fazer som e o arduino para controlar o sistema. A programação do projeto utiliza arrays para controlar todos os pinos dos leds e botões, além da sequência que deve ser seguida pelo jogador. E para verificar essa sequência são utilizados os loops(for).

Arduino, leds, botões e buzzer(mini alto-falante).
