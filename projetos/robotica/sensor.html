Sensor de Distância

Sobre o projeto

O sensor de distância que utiliza um sensor ultrassônico e uma programação no Arduino para criar rotinas para verificar a distância da placa de uma parede ou objeto e acender a cores verde, vermelho, azul sendo cada cor correspondente para uma distância.

Quem desenvolveu?

Adrian Cordeiro Piffer, Arthur Magalhães Ghirardi, Sara Gabriela de Oliveira.

Tecnologias utilizadas

Arduíno com sensor de distância ultrassônico e programação de rotinas
