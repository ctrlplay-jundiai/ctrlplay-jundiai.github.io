GEOMETRY DASH SQUARE PARKOUR

Sobre o projeto

projeto inspirado no jogo Geometry Dash, o jogo utiliza alguns objetos para dificultar a passagem do personagem, tendo uma programação de colisão com os espinhos para fazer o player perder e uma colisão com a bandeira para fazer o player ganhar

Quem desenvolveu?

teens 2 sábado 10h, Enzo Ribeiro, Enzo Jamariqueli Fonseca, Giovanne Consentino, Henrique Amorim Silva, Igor Antônio Andreaça Levada, Kauê Moreira dos Santos, Lucas Abreu de Souza, Lucas de Melo Pereira, Pietro de Oliveira Mantovani, Samuel Bertolucci Caetano, Guilherme de Paula rocha.

Tecnologias utilizadas

A plataforma utilizada foi o Game Maker, com programações voltadas à mudança de uma tela para a outra e com as colisões nos objetos para fazer o jogo ficar mais divertido e melhorar a experiência do jogador.
