FRUIT NINJA EXPLOSIVE FRUIT

Sobre o projeto

Esse projeto foi desenvolvido levando em base o jogo Fruit Ninja, o jogo utiliza spawners por que faz a fruta nascer aleatoriamente na tela, uma variável para contar os pontos, uma programação de eixo para a espada seguir o mouse no eixo x e y, contém também uma programação para iniciar o jogo quando o botão start é pressionado e uma programação para reiniciar o jogo caso tenha mais de 10 frutas na tela.

Quem desenvolveu?

teens 2 sábado às 13h, Davi Rodrigues Lima, Gabriel Henrique Urso, Isabela Rocha Barbosa Nicolau, Leonardo João dos Santos, Miguel da Silva Brandão.

Tecnologias utilizadas

A plataforma utilizada foi o Game Maker, utilizamos colisões para fazer as interações entre objetos e uma variável para atualizar nossa pontuação.
