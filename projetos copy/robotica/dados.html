Dado com Micro:bit

Sobre o projeto

O projeto “Dados para tabuleiro”, criado por turmas Kids, é um sistema que utiliza o micro:bit em combinação com as programações no make code para gerar a lógica de um dado, que gera números aleatórios que são utilizados para jogar com tabuleiro.

Quem desenvolveu?

Turmas Kids - Diversas

Tecnologias utilizadas

Micro:bit, tabuleiro, bateria.
