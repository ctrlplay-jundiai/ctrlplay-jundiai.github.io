CROSS ROADS

Sobre o projeto

Jogo 3D onde o objetivo principal é chegar o mais longe possível antes de morrer. Para controlar o personagem é utilizado o: W: ir para frente S: ir para trás A: ir para esquerda D: ir para direita

Quem desenvolveu?

Manoel Xavier de Oliveira Neto (Ctrl Teens 3)

Tecnologias utilizadas

GODOT: Ferramenta profissional de criação de jogos 2D e 3D para todos as plataformas como XBOX, PS4, PS5, Switch, PC, MAC, Android. A ferramenta usa a linguagem GML para a programação. MAGICAVOXEL: Ferramenta utilizada para criação de modelos 3D com voxel. Muito usada para fazer jogos com uma estética diferenciada e ou deixar os jogos mais otimizados.
